📝 Mini Sınav

10 soru · ~15 dk

8+ doğru = becerilerini güçlü pekiştirmişsin. 6–7 doğru = zayıf konuları tekrar et. 5 ve altı = Ders 21–25'i baştan gözden geçir.

1. Metnin genel fikrini hızlıca anlamak için hangi teknik?
  1. scan
  2. skim
  3. repeat
  4. draft
2. Belirli bir bilgiyi (ör. fiyat, tarih) bulmak için hangi teknik?
  1. skim
  2. scan
  3. outline
  4. hook
3. Paragrafın ana fikrini taşıyan cümle:
  1. conclusion
  2. topic sentence
  3. headline
  4. takeaway
4. "Bence bu iyi bir fikir" — hangisi?
  1. I think this is a good idea.
  2. I am think this is a good idea.
  3. This is think good idea.
  4. Think I this is good idea.
5. TED konuşmasında dinleyiciyi hemen yakalamak için kullanılan giriş:
  1. conclusion
  2. hook
  3. draft
  4. passage
6. Konuşmayı bitirirken "Özetlemek gerekirse…" — hangisi?
  1. To sum up
  2. To skim
  3. To repeat
  4. To scan
7. Dinleme pratiğinde telaffuzu taklit etmek için ne yaparsın?
  1. skim the audio
  2. repeat after the speaker
  3. write an outline
  4. turn left
8. "Muhtemelen yağmur yağacak" — hangisi?
  1. It will probably rain.
  2. It probably will raining.
  3. Probably it rains.
  4. It is probably rain.
9. Yazmaya başlamadan önce fikirleri düzenlemek için:
  1. write a draft first, no plan
  2. make an outline
  3. skim the paragraph
  4. use a hook
10. Sunum sonunda dinleyicinin aklında kalmasını istediğin ana mesaj:
  1. accent
  2. takeaway
  3. headline
  4. gist

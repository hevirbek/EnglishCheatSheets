Sayılabilen vs Sayılamayan — Temel Kural

Countableapple / apples
vs
Uncountablerice / water
✅ Sayılabilen (countable)
a/an · two · many · few
an apple · three eggs · many tomatoes
bir elma · üç yumurta · çok domates
💧 Sayılamayan (uncountable)
some · much · a little · a bottle of
some rice · much water · a glass of milk
biraz pirinç · çok su · bir bardak süt
Altın kural: Tek tek sayabiliyorsan → countable (an egg, two eggs). Sayamazsan → uncountable (some water, asla a water değil).
Ne sorarsın? Countable Uncountable
Ne kadar? How many apples? How much rice?
Tekil a / an apple ✗ a rice → some rice
Çoğul apples (s eklenir) rice (değişmez)
There is/are There are two eggs. There is some milk.

Some / Any — Kurallar

Cümle tipi Kural Örnek
Olumlu (+) some I have some apples.
There is some milk.
Olumsuz (−) any I don't have any eggs.
There isn't any cheese.
Soru (?) any (genel) Do you have any tomatoes?
Is there any bread?
Teklif / İstek some (özel!) Would you like some tea?
Can I have some water?
Teklif (offer)
Would you like + some + noun?
Would you like some coffee? — Biraz kahve ister misiniz?
İstek (request)
Can I have + some / a + noun?
Can I have some bread, please? · Can I have a menu?
Var mı? (question)
Do you have + any + noun?
Do you have any vegetarian dishes? — Vejetaryen yemek var mı?
İstisna: Teklif ve kibar isteklerde soru cümlesinde bile some kullanılır: Would you like some juice? (Cevap bekleniyor, teklif ediyorsun.)

Türkçe Konuşanların Klasik Tuzakları

🚨 How many water? — EN KRİTİK TUZAK
Su sayılamaz → how much. Elma sayılabilir → how many.
✗ How many water do you drink?
✓ How much water do you drink?
✓ How many apples do you eat?
informations / funs — çoğul ekleme
Information ve fun sayılamaz — asla -s eklenmez.
✗ I need some informations. · We had a lot of funs.
✓ I need some information. · We had a lot of fun.
Yemek siparişinde artikel unutmak
Türkçe'de "pizza istiyorum" derken İngilizce'de genelde a/the gerekir.
✗ I want pizza. · Can I have menu?
✓ I want a pizza. · Can I have the menu?
a water / two rices
Sayılamayan isimlerle a/an ve çoğul -s kullanılmaz. Konteyner kullan:
✗ a water · two rices · three breads
✓ a glass of water · some rice · two loaves of bread
Soru cümlesinde some kullanmak
Genel sorularda any; teklif/istek sorularında some:
✗ Do you have some milk? (genel soru)
✓ Do you have any milk? · Would you like some milk? (teklif)
✓ I'd like = kibar sipariş
Restoranda "I want" kaba gelebilir. Kibar alternatif:
I'd like the grilled chicken, please. · Can I have some more bread?

5 Dakikada Özet

🔢Countable: a/an, -s, how many, many.
💧Uncountable: some, how much, much — -s yok!
📦some = olumlu + teklif/istek. any = olumsuz + soru.
🍽️Sipariş: I'd like… · Can I have…?
👨‍🍳Pişirme: boil, fry, bake, chop, mix.
⚠️Tuzak: how many water ✗ → how much water ✓
🏆

Günün Görevi

📝
Restoran + Buzdolabı Pratiği 1. Bir restoran sipariş diyaloğu yaz (en az 8 replik). I'd like, Can I have, Would you like some ve Do you have any kullan.
2. Buzdolabındaki yiyecekleri listele — some/any ile:
There is some milk. · There aren't any eggs. · Do we have any cheese?
Bonus: Countable ve uncountable kelimeleri ayrı gruplara ayır.

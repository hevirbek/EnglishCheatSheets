Koşul Cümleleri (If Clauses)

B1 Gramer + Konuşma ⏱ ~2 saat
Türkçe tuzak: If'ten sonra will yok — present kullan.
✗ If I will go…
✓ If I go, I will…
Gerçek gelecek = 1st · Hayali = 2nd · Geçmiş pişmanlık = 3rd
🔁 Ders 17 although = rağmen (zıtlık) however = ancak (cümle başı) therefore = bu yüzden (sonuç) Although it rained, we went out. · It rained. However, we went out.
🔬

Zero Conditional — Genel Gerçekler / Bilim

If + presentpresent simple
,
presentpresent simple
Ne zaman? Her zaman doğru olan gerçekler, bilimsel kurallar, otomatik sonuçlar. Zaman fark etmez — genel doğa yasası gibi düşün.
🔬 Bilim & genel gerçek
If + present, present
If you heat ice, it melts.
If water reaches 100°C, it boils.
If I don't sleep enough, I feel tired.
Buzu ısıtırsan erir. · Su 100°C'ye ulaşırsa kaynar. · Yeterince uyumazsam yorgun hissederim.
When = If: Genel gerçeklerde when de kullanılabilir: When I wake up, I drink coffee.

First Conditional — Gerçek Gelecek

If + presentpresent simple
,
will + basegelecek
Ne zaman? Koşul gerçekleşirse sonuç gerçekten olacak. Olasılık yüksek veya makul — plan, tehdit, vaat, tahmin.
🎯 Gerçek gelecek olasılık
If + present, will + base
If it rains tomorrow, we will stay home.
If you study hard, you will pass the exam.
I'll call you if I arrive early.
Yarın yağmur yağarsa evde kalırız. · Sıkı çalışırsan sınavı geçersin. · Erken varırsam seni ararım.
Alternatifler: will yerine can / may / might / should kullanılabilir:
If you finish early, you can leave. · If he's late, he might miss the train.
💭

Second Conditional — Hayali / Gerçek Dışı Şimdi

If + past simplewas/were
,
would + basehayali sonuç
Ne zaman? Şu an gerçek olmayan veya çok düşük olasılıklı durumlar. Hayal, tavsiye, nazik istek, eleştiri.
💭 Hayali şimdi / düşük olasılık
If + past simple, would + base
If I had a million dollars, I would travel the world.
If I were you, I would accept the offer.
What would you do if you won the lottery?
Bir milyon dolarım olsaydı dünyayı gezerdim. · Senin yerinde olsam teklifi kabul ederdim. · Piyangoyu kazansan ne yapardın?
If I were… Resmi/kurallı İngilizcede tüm öznelerle were tercih edilir: If I were rich… (If I was rich… konuşmada duyulur ama sınavda were güvenli)
⏪

Third Conditional — Geçmiş Pişmanlık (Kısa Giriş)

If + past perfecthad + V3
,
would have + V3hayali geçmiş
Ne zaman? Geçmişte olan bir şeyin tersini hayal etmek — pişmanlık, eleştiri, "keşke" cümleleri. Koşul gerçekleşmedi, sonuç da olmadı.
⏪ Geçmişte olmayan — pişmanlık
If + had + V3, would have + V3
If I had studied harder, I would have passed.
If she had left earlier, she wouldn't have missed the flight.
We would have arrived on time if we hadn't taken the wrong turn.
Daha sıkı çalışsaydım geçerdim. · Erken çıksaydı uçağı kaçırmazdı. · Yanlış dönmeseydik zamanında varırdık.
B1 notu: Third conditional detaylı çalışma Ders 19'da devam eder. Şimdilik yapıyı tanı — had + V3 + would have + V3 eşleşmesi kritik.

Unless · As Long As · In Case

🚫 Unless = if … not
unless + present (1st) · unless + past (2nd)
I won't go unless you come with me.
= I won't go if you don't come with me.
Unless it stops raining, we won't leave.
Sen gelmezsen gitmem. · Yağmur durmazsa çıkmayız.
✅ As long as = provided that
koşul sağlandığı sürece
You can stay as long as you keep quiet.
As long as you finish by Friday, it's fine.
Sessiz kaldığın sürece kalabilirsin. · Cuma'ya kadar bitirirsen sorun yok.
🛡 In case = as a precaution
ihtimaline karşı · -mesi ihtimaline karşı
Take an umbrella in case it rains.
I'll save some food in case you're hungry later.
Yağmur yağması ihtimaline karşı şemsiye al. · Sonra acıkman ihtimaline karşı yemek ayırırım.
Unless tuzak: Unless'ten sonra not kullanma — zaten olumsuz anlam taşır.
✗ Unless you don't hurry…  →  ✓ Unless you hurry…

Türkçe Konuşanların Klasik Tuzakları

🚨 If I will go — EN KRİTİK TUZAK
Türkçe "-ecek" gelecek zaman if cümlesine taşınır. If'ten sonra asla will kullanma — present simple kullan.
✗ If I will go to London, I will visit Big Ben.
✓ If I go to London, I will visit Big Ben.
Will vs Would karıştırmak (1st vs 2nd)
Gerçek gelecek = will. Hayali/gerçek dışı = would. Aynı cümlede ikisini karıştırma.
✗ If I win the lottery, I would buy a house. (gerçek gelecek için)
✓ If I win the lottery, I will buy a house. (1st — olabilir)
✓ If I won the lottery, I would buy a house. (2nd — hayal)
Virgül kuralı
If cümlesi başta gelirse virgül kullan. Sonda gelirse virgül yok.
✓ If it rains, we will stay home.
✓ We will stay home if it rains. (virgül yok)
Unless + not
Unless zaten "if not" demek — çift olumsuz yapma.
✗ Unless you don't call me…
✓ Unless you call me… = If you don't call me…
✓ In case ≠ If
In case = ihtimaline karşı önlem al. If = koşul gerçekleşirse.
Take a jacket in case it gets cold. (önlem — henüz soğuk değil)
If it gets cold, I'll put on my jacket. (koşul gerçekleşince)

Gerçek Hayat Diyaloğu — Seyahat Planlama

🔵 Mavi = 1st conditional (will)  ·  🟣 Mor = 2nd conditional (would)
A
Have you thought about our trip to Italy? If we book early, we'll get cheaper flights.
İtalya seyahatimizi düşündün mü? Erken rezervasyon yaparsak daha ucuz uçak bileti buluruz.
B
True. But if I had more vacation days, I would stay for two weeks instead of one.
Doğru. Ama daha fazla tatil günüm olsaydı bir yerine iki hafta kalırdım.
A
If the weather is good, we can visit the Amalfi Coast. We should book a hotel near the station too.
Hava güzelse Amalfi Kıyıları'nı ziyaret edebiliriz. İstasyon yakınına otel de ayırtmalıyız.
B
If I were you, I would choose a place with a kitchen. Eating out every day is expensive.
Senin yerinde olsam mutfaklı bir yer seçerdim. Her gün dışarıda yemek pahalı.
A
Good idea. If we don't find a good hotel, we'll stay in an Airbnb. I'll also pack a power adapter in case the sockets are different.
İyi fikir. İyi otel bulamazsak Airbnb'de kalırız. Prizler farklı olması ihtimaline karşı adaptör de koyacağım.
B
If you send me the dates today, I'll check the flights tonight. What would you do if we missed the last train to Rome?
Bugün tarihleri gönderirsen bu gece uçuşlara bakarım. Roma'ya son trene yetişemezsek ne yapardın?
A
If that happened, I would take a taxi. But if we leave the airport by 6, we won't miss it.
Öyle olsaydı taksi tutardım. Ama havaalanından saat 6'da çıkarsak kaçırmayız.

Pratik — Kendini Test Et

A) Zero / 1st / 2nd — doğru tipi seç
1. If you mix vinegar and baking soda, it (fizz). → zero
2. If she (call) tonight, tell her I'm busy. → 1st
3. If I (be) rich, I (buy) a yacht. → 2nd
4. Water (freeze) if the temperature drops below 0°C. → zero
5. If it (rain) tomorrow, the match (be) cancelled. → 1st
Cevapları göster
1. fizzes   2. calls   3. were/was, would buy   4. freezes   5. rains, will be
B) Unless / as long as / in case
1. You can't enter you show your ID. (if not)
2. I'll lend you my car you drive carefully. (provided that)
3. Take some cash the card machine doesn't work.
4. We won't start everyone is here. (if not)
Cevapları göster
1. unless   2. as long as   3. in case   4. unless
C) Hatayı bul ve düzelt
1. If I will see him, I'll say hello. →
2. If I would have time, I will help you. →
3. Unless you don't hurry, you'll be late. →
4. If she wins, she would donate the money. →
Cevapları göster
1. If I see him, I'll say hello.   2. If I have time, I will help you. / If I had time, I would help you.   3. Unless you hurry, you'll be late.   4. If she wins, she will donate the money. / If she won, she would donate the money.
D) Cümle tamamla — 1st veya 2nd
1. If I find a cheap flight, I (book) it immediately.
2. If I (speak) fluent English, I (apply) for that job abroad.
3. What you (do) if your phone (break) tomorrow?
4. If they (offer) me the position, I (accept) it. (gerçek olasılık)
Cevapları göster
1. will book / 'll book   2. spoke, would apply   3. will you do, breaks (1st) / would you do, broke (2nd — ikisi de kabul)   4. offer, will accept / 'll accept

5 Dakikada Özet

🔬Zero: If + present, present — genel gerçekler, bilim.
🎯First: If + present, will + base — gerçek gelecek. If'ten sonra will yok!
💭Second: If + past, would + base — hayali şimdi. If I were you…
⏪Third: If + had + V3, would have + V3 — geçmiş pişmanlık.
🚫Unless = if not · As long as = koşul sağlandığı sürece · In case = ihtimaline karşı.
✏️If başta → virgül. If sonda → virgül yok.

Günün Görevi

Hayatına Dair Koşul Cümleleri — 9 Cümle Kendi hayatın hakkında aşağıdaki koşul tiplerini kullanarak her tip için 3 cümle yaz:

1st conditional (3 cümle): Gerçek gelecek planların
Örn: If I save enough money, I will visit Japan next year.

2nd conditional (3 cümle): Hayali durumlar / "keşke" senaryoları
Örn: If I didn't have to work on Saturdays, I would learn to play the guitar.

3rd conditional (3 cümle): Geçmiş pişmanlıklar
Örn: If I had started English earlier, I would have been fluent by now.
Bonus: Cümlelerini yüksek sesle oku ve if cümlesi başta mı sonda mı olduğuna göre virgülü kontrol et.
